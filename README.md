## SUMMARY
This project demonstrates the ideas of [Mapped Diagnostic Context] using [Spring Boot]
and [Logback]

## WHERE TO START

### Prerequisites
- [Gradle](https://gradle.org/install)
- JDK

### Libraries
- Spring Boot
- Logback
